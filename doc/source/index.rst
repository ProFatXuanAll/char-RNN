Welcome to Language Model Playground's documentation!
=====================================================

Language Model Playground is based on Pytorch. 
It contains several models including RNN, LSTM, GRU, Attention.
Language Model Playground is able to:

    * Generate sentences
    * Evaluate models
    * Train tokenizers
    * Train neural network based language model
    * Generate text using language model
    * And more to come. 

Get started with :doc:`Quickstart <quickstart>` .


Language Model Playground depends on several toolkits, and the document is generated by Sphinx

    * `Pytorch <https://pytorch.org/docs/stable/index.html>`_ 
    * `Sphinx <https://www.sphinx-doc.org/en/master/>`_ 
    * `Sklearn <https://scikit-learn.org/>`_
