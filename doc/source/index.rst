Welcome to Language Model Playground's documentation!
=====================================================

Language Model Playground is based on Pytorch. It contains several models including RNN, LSTM, GRU, Attention,
and it can generate sentences, evaluation models, inference words. 
Get started with :ref:`my-reference-label`.


Language Model Playground depends on several toolkits, and the document is generated by Sphinx

    * `Pytorch <https://pytorch.org/docs/stable/index.html>`_ 
    * `Sphinx <https://www.sphinx-doc.org/en/master/>`_ 
    * `Sklearn <https://scikit-learn.org/>`_
